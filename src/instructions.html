This task will explore your understanding of NodeJS and its features

There are two endpoints in this NodeJS program

  1. /health - Will return "good" as response
  2. /test - Will calculate a complicated Hash and return it

Follow the below steps to know what problem needs to solved

  1. First open the "/health" endpoint and confirm that it return "good".
  2. Now open the "/test" endpoint and then in new tab open the "/health" enpoint.
  3. You will notice that "/health" enpoint will wait until "/test" endpoint has finished its processing.

Your task is to make changes in the code so that "/health" returns the response even if "/test" is still calculating the hashes.
